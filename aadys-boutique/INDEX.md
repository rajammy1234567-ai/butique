## ⚡ Quick Start (5 Minutes)

### Prerequisites
- Node.js installed
- MongoDB Atlas account (free)
- Razorpay account (free)

### Steps
```bash
# 1. Setup environment
# Follow SETUP.md for .env configuration

# 2. Install backend
cd backend
npm install
npm run dev

# 3. Install frontend (new terminal)
cd frontend
npm install
npm run dev

# 4. Open browser
# http://localhost:5173
```
