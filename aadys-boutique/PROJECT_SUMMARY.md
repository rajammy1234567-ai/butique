## 🚀 Quick Start (5 Minutes)

### 1. Get Credentials
```
MongoDB URI: mongodb+srv://user:pass@cluster.mongodb.net/aadys_boutique
Razorpay Key ID: from dashboard
Razorpay Key Secret: from dashboard
```

### 2. Backend Setup
```bash
cd aadys-boutique/backend
npm install
cp .env.example .env
# Edit .env with credentials
npm run dev
```

### 3. Frontend Setup
```bash
cd ../frontend
npm install
cp .env.example .env
npm run dev
```

### 4. Open Browser
```
Frontend: http://localhost:5173
Backend: http://localhost:5000
Admin: http://localhost:5173/admin/dashboard
```
