## 🚀 3 Steps to Launch

### Step 1: Setup (30 minutes)
```bash
1. Install Node.js from nodejs.org
2. Create MongoDB Atlas account (free)
3. Create Razorpay account (free)
4. Copy connection strings to .env files
5. Run: npm install in backend and frontend
```

### Step 2: Start (2 minutes)
```bash
# Terminal 1 - Backend
cd backend
npm run dev

# Terminal 2 - Frontend  
cd frontend
npm run dev

# Open browser: http://localhost:5173
```

### Step 3: Test (10 minutes)
```bash
1. Register as user
2. Make admin (instructions in SETUP.md)
3. Add test products via admin
4. Test payment with 4111 1111 1111 1111
5. Check order in admin panel
```
