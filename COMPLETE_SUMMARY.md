## Getting Started - Next 30 Minutes

### Step 1: Download Files (2 minutes)
1. Download folder: `aadys-boutique-complete/` OR
2. Download compressed: `aadys-boutique-complete.tar.gz`

### Step 2: Get Credentials (10 minutes)

**MongoDB Atlas (Free - 25 GB)**
- Visit: mongodb.com/cloud/atlas
- Create account → Cluster → Get connection string

**Razorpay (Free - Test Mode)**
- Visit: razorpay.com
- Create account → API Keys → Copy keys

**Cloudinary (Free - 25 GB)**
- Visit: cloudinary.com
- Create account → Dashboard → Copy credentials

**JWT Secret**
- Generate: Random 32+ character string

### Step 3: Setup Backend (5 minutes)
```bash
cd backend
npm install
cp .env.example .env
# Edit .env with your credentials
npm run dev
# Backend runs on http://localhost:5000
```

### Step 4: Setup Frontend (5 minutes)
```bash
cd ../frontend
npm install
cp .env.example .env
# Edit .env with your credentials
npm run dev
# Frontend runs on http://localhost:5173
```

### Step 5: Test (8 minutes)
- Open http://localhost:5173
- Login with phone: 9029411841
- Browse products
- Add to cart
- Checkout with test payment
- Check admin panel
